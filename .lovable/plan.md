

## שדרוג שדה "סוג" + תצוגה מקדימה של פייסבוק

### שינוי 1 — סוג הנכס: השכרה / מכירה

כרגע בשדה "סוג" (במצב אוטומטי חוזר) יש רק "דירות (מחזורי)" ו"כתבות (חד-פעמי)". המשתמש רוצה לבחור גם **סוג עסקה** — השכרה או מכירה — כדי שהתור יפרסם רק דירות רלוונטיות.

**פתרון**: הוספת dropdown נוסף "סוג נכס" עם אפשרויות:
- הכל (ברירת מחדל)
- השכרה בלבד
- מכירה בלבד

הערך יישמר בשדה חדש `property_filter` ב-`auto_publish_queues` (או כ-JSON בשדה קיים). בצד הטופס — dropdown פשוט שמופיע כש-`queueType === 'property_rotation'`.

גם ב**מצב חד-פעמי** — חלוקת רשימת הנכסים ל-2 קבוצות (optgroup) — "להשכרה" ו"למכירה" — כדי שיהיה קל יותר למצוא.

### שינוי 2 — תצוגה מקדימה בסגנון פייסבוק

הוספת Preview שנראה כמו פוסט אמיתי בפייסבוק: avatar + שם דף, תאריך, טקסט הפוסט (אחרי החלפת placeholders), תמונות (grid), והאשטגים. מוצג מתחת ל-textarea, מתעדכן בזמן אמת.

```text
┌──────────────────────────────────────┐
│ [🔵] PrimeProperty         עכשיו    │
│                                      │
│ 🏠 דירה להשכרה ב תל אביב            │
│ 📍 הרצל 10, לב העיר                  │
│ 💰 ₪5,500 לחודש                      │
│ ...                                  │
│                                      │
│ ┌──────────────────────────────────┐ │
│ │         [תמונת הנכס]            │ │
│ └──────────────────────────────────┘ │
│                                      │
│ #נדלן #דירהלהשכרה #תלאביב           │
│ 👍 💬 ↗️                              │
└──────────────────────────────────────┘
```

### שינויים טכניים

| קובץ | שינוי |
|-------|-------|
| Migration SQL | `ALTER TABLE auto_publish_queues ADD COLUMN property_filter TEXT DEFAULT 'all';` |
| `AutoPublishManager.tsx` | 1. הוספת state `propertyFilter` + dropdown "סוג נכס" (הכל/השכרה/מכירה) במצב recurring 2. סינון רשימת נכסים לפי `property_type` במצב one_time (optgroups) 3. הוספת רכיב FacebookPreview inline מתחת ל-textarea — כרטיס עם עיצוב פייסבוק (border radius, shadow, avatar, reactions bar) שמציג את הטקסט + תמונות + האשטגים |
| `useAutoPublish.ts` | עדכון mutation לשמור `property_filter` |
| Edge Function `auto-publish` | סינון דירות לפי `property_filter` בעת בחירת הדירה הבאה |
