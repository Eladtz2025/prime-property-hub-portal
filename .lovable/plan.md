

## שדרוג מערכת פרסום — קומפקטית ומתקדמת

### שינויים עיקריים

**1. הסרת טאב הגדרות → אייקון הגדרות**
- טאב "הגדרות" יוסר מה-TabsList
- במקומו: אייקון ⚙️ בפינה השמאלית של הכותרת שפותח `Sheet` (מגירה צדדית) עם כל ההגדרות + סטטוס חיבור
- חיסכון: מ-5 טאבים ל-4

**2. איחוד קבוצות + תבניות לטאב אחד**
- טאב "כלים" חדש שמכיל קבוצות ותבניות בתוך Accordion/Collapsible
- מ-4 טאבים ל-3: "פוסט חדש" | "היסטוריה" | "כלים"
- פחות טאבים = יותר קומפקטי

**3. כרטיסי סיכום — שורה אחת דחוסה**
- במקום 4 כרטיסים גדולים: שורת inline badges קומפקטית
- דוגמה: `✅ 0 פורסמו  ·  ⏰ 0 מתוזמנים  ·  📝 0 טיוטות  ·  📨 0 סה"כ`
- חיסכון משמעותי בגובה

**4. Composer — יותר קומפקטי**
- הסרת CardHeader מיותר ("יצירת פוסט חדש" — ברור מההקשר)
- כפתורי פלטפורמה + בחירת מקור/תבנית באותה שורה
- אזור תמונות: grid קטן יותר (4 עמודות)
- תצוגה מקדימה: רק כש-preview panel פתוח (toggle), לא תמיד מוצגת — חוסך 40% מהרוחב

**5. היסטוריה — קומפקטית יותר**
- הסרת כותרת CardHeader (הטאב כבר אומר "היסטוריה")
- פילטרים: inline בשורה אחת עם הכותרת
- כרטיסי פוסטים: padding קטן יותר, טקסט 11px
