# ✅ הושלם: הסתרת נכסים לא פעילים + שיפור השלמת נתונים

## שינויים שבוצעו

### 1. פילטר `is_active = true` נוסף לכל השאילתות
- `applyFilters` - כל חיפוש בטבלה
- סטטיסטיקות: totalCount, todayCount, weekCount, bySources
- שאילתת neighborhoods

### 2. שיפור Edge Function - זיהוי tasks תקועים
- אם task רץ יותר מ-10 דקות בלי עדכון → מסומן כ-stopped אוטומטית
- הלחיצה הבאה על הכפתור תתחיל task חדש

## תוצאה
- רק נכסים פעילים מוצגים בטבלה
- כפתור השלמת נתונים מתאושש מתקיעות
