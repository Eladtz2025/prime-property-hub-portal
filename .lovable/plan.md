
# תיקון חילוץ מחירים - extractPrice ✅ DONE

## מה תוקן
`extractPrice` ב-`parser-utils.ts` מחק את **כל הרווחים** לפני חילוץ, מה שגרם למספר החדרים להתמזג עם המחיר:
- `3,980,000₪ 3 חד׳` → אחרי הסרת רווחים/₪/פסיקים: `39800003` → מחיר שגוי

## הפתרון
שלושה שלבי חילוץ עם ₪ כעוגן (בלי למחוק רווחים!):
1. **Before ₪**: `/([\d,]+)\s*₪/` - מספר לפני סימן שקל
2. **After ₪**: `/₪\s*([\d,]+)/` - מספר אחרי סימן שקל  
3. **ש"ח**: `/([\d,]+)\s*ש"ח/` - מספר לפני ש"ח
