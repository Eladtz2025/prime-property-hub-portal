
# תיקון דף הפרויקטים באנגלית

## הבעיות
1. שמות הפרויקטים מוצגים בעברית כי `title_en` ריק בDB
2. העיר מוצגת בעברית (תל אביב-יפו) -- אין שדה `city_en` ואין תרגום
3. אין שליפת יחידות (units) בדף האנגלי -- אז אין טבלת יחידות ואין סיכום זמינות

## פתרון

### 1. עדכון שמות אנגליים בDB
הרצת SQL שמעדכן את `title_en` לשלושת הפרויקטים:
- סוקולוב 23 -> "Sokolov 23"
- זלטופולסקי 27 -> "Zlatopolsky 27"  
- עמוס -> "Amos"

### 2. תרגום עיר בצד הקליינט
בדף האנגלי, החלפת שם העיר לאנגלית לפני העברה לכרטיס:
- "תל אביב-יפו" -> "Tel Aviv"
- "ירושלים" -> "Jerusalem"
- ברירת מחדל: שם העיר כמו שהוא

### 3. הוספת שליפת יחידות לדף האנגלי
העתקת הלוגיקה מהדף העברי -- שליפת `project_units` והעברת `units` prop לכרטיסים, כך שתוצג טבלת יחידות וסיכום זמינות.

## קבצים לשינוי
1. `src/pages/en/NewDevelopments.tsx` -- תרגום עיר + שליפת units
2. הרצת SQL לעדכון `title_en`
