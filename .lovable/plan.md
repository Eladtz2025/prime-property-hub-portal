
# תיקון חילוץ מחירים - Format D (Jina) ✅ DONE

## מה תוקן
חילוץ מחירים ב-Format D (Jina single-line) שבו מחיר + חדרים מתמזגים (e.g. `80,000,001` = `8,000,000` + `1` חדר).

## הפתרון שיושם
שלושה שלבי חילוץ בסדר עדיפות:
1. **Combined regex**: `/([\d,]+)\s*₪\s*(\d+\.?\d*)\s*חד[׳'ר]/` - תופס מחיר + חדרים בבת אחת
2. **Strict comma format**: `/(d{1,3}(?:,\d{3})+)\s*₪/` - רק מספרים עם פסיקים תקינים
3. **Simple with sanity check**: max 50M למכירה, max 100K להשכרה
