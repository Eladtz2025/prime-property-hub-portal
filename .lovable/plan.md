

# תיקון מיפוי השכונות למדלן - רשימה מדויקת

## הבעיות שזוהו

לאחר בדיקה מול אתר מדלן, הקובץ הנוכחי מכיל טעויות:

| שכונה | מה רשום | מה חסר/שגוי |
|-------|---------|-------------|
| צפון ישן | 3 תת-אזורים | חסר **דרום מערבי** (יש 4!) |
| תל ברוך | תת-אזור אחד | חסר **תל ברוך צפון** |
| רמת אביב | רק "החדשה" | חסר **רמת אביב** הרגיל |
| מרכז העיר | `לב-תל-אביב-לב-העיר-צפון` | שם לא נכון |

---

## הרשימה המדויקת מאתר מדלן

### צפון ישן (4 תת-אזורים)
```
שכונה-הצפון-הישן-החלק-הצפוני
שכונה-הצפון-הישן-החלק-המרכזי
שכונה-הצפון-הישן-החלק-הדרום-מזרחי
שכונה-הצפון-הישן-החלק-הדרום-מערבי  ← חדש!
```

### צפון חדש (3 תת-אזורים - נכון)
```
שכונה-הצפון-החדש-החלק-הצפוני
שכונה-הצפון-החדש-החלק-הדרומי
שכונה-הצפון-החדש-סביבת-כיכר-המדינה
```

### תל ברוך (2 תת-אזורים)
```
שכונה-תל-ברוך
שכונה-תל-ברוך-צפון  ← חדש!
```

### רמת אביב (2 תת-אזורים)
```
שכונה-רמת-אביב  ← הוספה
שכונה-רמת-אביב-החדשה
```

### מרכז העיר / לב תל אביב
```
שכונה-לב-תל-אביב  ← תיקון
```

## סיכום השינויים

| שכונה | שינוי |
|-------|-------|
| צפון ישן | +1 תת-אזור (דרום מערבי) |
| תל ברוך | +1 תת-אזור (צפון) |
| רמת אביב | +1 תת-אזור (רמת אביב הרגיל) |
| מרכז העיר | תיקון שם slug |
| לב תל אביב | הוספת alias |

---

## בדיקה אחרי התיקון

1. להריץ סריקת מדלן עם צפון ישן
2. לוודא בלוגים שה-URL כולל 4 תת-אזורים (כולל דרום-מערבי)
3. לוודא שיש ~150 דירות בתוצאות (לא פחות מ-100)
