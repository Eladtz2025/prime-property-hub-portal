

## פיצול לשני עמודות: טופס מימין + תצוגה מקדימה משמאל

### מצב נוכחי
כל הטופס והתצוגה המקדימה מוצגים אנכית — אחד מתחת לשני. זה גורם לגלילה מיותרת.

### שינוי

**קובץ: `AutoPublishManager.tsx`**

עטיפת ה-`CardContent` בתוך `grid grid-cols-2 gap-4`:
- **עמודה ימנית** — כל הבקרות: שורת סוג/נכס/תבנית/פלטפורמה, textarea, האשטגים, סגנון פוסט + גלריית תמונות, תזמון, כפתורי פעולה
- **עמודה שמאלית** — `FacebookPostPreview` בלבד (sticky כדי שתמיד נראית)

### מבנה

```text
┌──────────────────────────────────────────────────────────┐
│ [חד-פעמי▾] [בחר נכס▾] [תבנית▾] [FB][IG] [עמוד][קבוצות]│
├────────────────────────────┬─────────────────────────────┤
│ ┌────────────────────────┐ │  ┌─────────────────────┐    │
│ │ כתוב תוכן הפוסט...    │ │  │   Facebook Preview  │    │
│ │                        │ │  │                     │    │
│ └────────────────────────┘ │  │   (sticky top)      │    │
│ [#האשטגים]                 │  │                     │    │
│ 🔗Link | 🖼תמונות          │  └─────────────────────┘    │
│ [img][img][img][img]       │                             │
│ [📅 תאריך] [10:00]        │                             │
│ 🔒פרטי [פרסם] [תזמן]      │                             │
├────────────────────────────┴─────────────────────────────┤
```

### פירוט טכני

1. שורת הבקרות (סוג פרסום, נכס, תבנית, פלטפורמה, יעד) **נשארת full-width מעל הגריד** — `col-span-2`
2. מתחת — `grid grid-cols-2 gap-4`:
   - Col 1 (ימין): textarea, hashtags, post style, image gallery, scheduling, actions
   - Col 2 (שמאל): `FacebookPostPreview` עטוף ב-`sticky top-4`
3. קבוצות פייסבוק (אם נבחר "קבוצות") יישאר full-width מעל הגריד

**קובץ אחד: `AutoPublishManager.tsx`**
