

## תיקון פרופורציות עמוד Insights

### הבעיות שרואים בצילומי מסך
1. **כרטיס כתבה בודד נדחק לצד ימין** — grid של 2 עמודות עם פריט אחד = חצי עמוד ריק משמאל
2. **Aspect ratio 3:4 גבוה מדי** — הכרטיס תופס יותר מדי גובה מסך, לא מתאים לעמוד רשימה (מתאים ל-division cards קטנים, לא לכרטיס בודד ענק)
3. **סקשן מדריכים ריק** — empty state עם הרבה רווח לבן מיותר
4. **חוסר איזון כללי** — הכרטיס הגדול + empty state הריק = העמוד מרגיש חלול

### פתרון

**כשיש כתבה אחת בלבד** — layout אופקי (תמונה + טקסט צמודים, לא כרטיס אנכי):
- Grid של 2 עמודות: תמונה בצד ימין, תוכן בצד שמאל
- Aspect ratio `4:3` (רוחבי) במקום `3:4` (אנכי)
- הכרטיס תופס את כל הרוחב

**כשיש 2+ כתבות** — כרטיסים אנכיים עם aspect ratio `4:5` (מעט פחות גבוה מ-3:4)

**סקשן מדריכים ריק** — הסתר לגמרי כשאין מדריכים (במקום להציג empty state שתופס מקום)

**סקשן בעלי מקצוע** — קצר את ה-padding כשהעמוד קצר
