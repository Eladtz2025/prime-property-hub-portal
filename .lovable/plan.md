

## מסגרת כהה לכל סקשן בדשבורד

### הרעיון

כל אחד מ-3 הסקשנים (פגישות, רעיונות, פניות) יקבל **מסגרת/wrapper בצבע הכהה של ההדר** (`hsl(220 20% 18%)` — ה-primary). הכרטיס הלבן יישב בתוך המסגרת הכהה, מה שייצור את הקונטרסט והתחושה של "עולם בפני עצמו" כמו בתמונת הרפרנס.

### מבנה

```text
┌─────────────────── dark frame (primary bg, rounded, padding) ───┐
│                                                                  │
│   ┌──────────── white card (existing) ────────────┐             │
│   │  Header + Content                              │             │
│   └─────────────────────────────────────────────────┘             │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### שינויים

**`Dashboard.tsx`** — עטיפת כל אחד מ-3 הכרטיסים ב-div עם:
- `bg-primary rounded-xl p-2 shadow-lg` — רקע כהה, פינות מעוגלות, padding קטן, צל
- הכרטיס הפנימי נשאר כמו שהוא (לבן עם ה-borders והעיצוב שכבר הוספנו)

**`UpcomingAppointmentsCard.tsx`** — ללא שינוי (הכרטיס הפנימי נשאר)

**`DevelopmentIdeasCard.tsx`** — ללא שינוי

כל השינוי הוא בעצם 3 div wrappers חדשים ב-`Dashboard.tsx` בלבד.
