

# שני כרטיסים זה לצד זה -- זמנים + ריצות אחרונות

## מה ישתנה

במקום טאבים בתוך כרטיס אחד, יהיו **שני כרטיסים נפרדים זה לצד זה** (grid של 2 עמודות):

```text
+---------------------------+---------------------------+
|  לוח זמנים (קיים)         |  ריצות אחרונות             |
|                           |                           |
|  00:00 - 02:30  כפילויות  |  00:12  כפילויות  8 דק׳   |
|  03:00 - 04:30  השלמה     |    4,574 נבדקו, 390 כפילויות|
|  05:00 - 06:30  זמינות    |  03:08  השלמה  37 דק׳      |
|  07:00 - 08:30  התאמות    |    260 עובדו, 235 הצליחו   |
|  23:00  יד2 השכרה ...     |  05:02  זמינות  6 דק׳      |
|                           |    18 נבדקו, 7 לא זמינים   |
+---------------------------+---------------------------+
```

## פרטים טכניים

### קובץ: `src/components/scout/ScheduleSummaryCard.tsx`

1. הסרת מנגנון הטאבים (state של activeTab, הכפתורים)
2. שינוי ה-return ל-`grid grid-cols-1 md:grid-cols-2 gap-4` עם שני Card נפרדים:
   - **כרטיס שמאל**: לוח הזמנים -- בדיוק כמו שהוא היום (ללא שינוי בתוכן)
   - **כרטיס ימין**: ריצות אחרונות -- מציג את הריצות ממוינות לפי שעה (יורד), כל שורה עם: נקודת צבע + אייקון סטטוס, שם התהליך, שעה, משך, וסיכום
3. ה-query של הריצות יהיה תמיד enabled (לא תלוי בטאב)
4. הלגנדה תישאר משותפת מעל שני הכרטיסים
