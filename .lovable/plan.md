

# הוספת תמונת הדמיה לכרטיסיית פרויקט

## מצב נוכחי
- לפרויקטים אין כרגע תמונות בטבלת `property_images`
- הכרטיסייה מציגה רק טקסט על רקע גרדיאנט כהה

## מה ישתנה
- תמונת ההדמיה הראשית תוצג **מעל** תוכן הכרטיסייה (חלק עליון)
- אם אין תמונה, הכרטיסייה תישאר כמו היום (גרדיאנט בלבד)

## שינויים

### 1. שליפת תמונות (`NewDevelopments.tsx` - HE + EN)
- הוספת query שמביא את התמונה הראשית (`is_main = true` או הראשונה לפי `order_index`) לכל פרויקט
- העברת `imageUrl` כ-prop ל-`NewDevelopmentCard`

### 2. הצגת תמונה בכרטיס (`NewDevelopmentCard.tsx`)
- הוספת prop `imageUrl?: string`
- מעל אזור התוכן הקדמי: הצגת תמונה עם aspect ratio 16:9
- שימוש ב-`OptimizedImage` עם lazy loading
- אם אין תמונה -- לא מוצג כלום, הכרטיס נשאר כמו היום

### עיצוב
```text
+---------------------------+
|                           |
|    [תמונת הדמיה 16:9]     |
|                           |
+---------------------------+
|  סטטוס                    |
|  שם הפרויקט               |
|  שכונה, עיר               |
|  מפרט + יחידות            |
|  לקבלת מידע →             |
+---------------------------+
```

## קבצים לשינוי

2. `src/pages/he/NewDevelopments.tsx` -- שליפת תמונות והעברה לכרטיס
3. `src/pages/en/NewDevelopments.tsx` -- אותו דבר לגרסה האנגלית
